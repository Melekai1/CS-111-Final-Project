# Movie Analyzer

WHAT IS IT: A C++ application that analyzes box office data for a variety of films

HOW TO RUN
 - Ensure that VS Code is installed with along C++ Extension Pack
 - Open a new Terminal and type the following commands:
     - g++ main.cpp library.cpp -o main
     - ./main
 - When asked for the name of the input file, enter "input.dat"

FEATURES
 - Display data
 - Search for a specific movie
 - Sort the list of movies in alphabetical order
 - Find the country with the highest gross amount for a specific movie
 - Gather data from movies distributed by a specific studio
 - Find the movie with highest domestic box office gross
 - Remove a movie from the list
